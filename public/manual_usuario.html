1. Introdução

Bem-vindo ao CRM PEEEM, um sistema completo para gestão de salões de beleza e estética. Este manual fornece informações detalhadas sobre todas as funcionalidades do sistema, ajudando você a aproveitar ao máximo os recursos disponíveis.

O CRM PEEEM foi desenvolvido para simplificar e otimizar a gestão do seu negócio, permitindo controlar agendamentos, gerenciar clientes, controlar finanças, administrar estoque de produtos e muito mais.

2. Primeiros Passos

2.1. Acesso ao Sistema

Para acessar o CRM PEEEM, siga os passos abaixo:

  1. Abra seu navegador preferido (recomendamos Chrome, Firefox, Edge ou Safari mais recentes)
  2. Digite o endereço do sistema na barra de navegação
  3. Na tela de login, digite seu e-mail e senha cadastrados
  4. Clique no botão "Entrar"
Dica: Ao acessar o sistema pela primeira vez, use as credenciais fornecidas pelo administrador. Recomendamos alterar sua senha no primeiro acesso.

2.2. Interface Principal

A interface do CRM PEEEM é composta pelos seguintes elementos:

  • Barra lateral de navegação: Localizada à esquerda, permite acesso rápido às principais seções do sistema.
  • Barra superior: Contém o acesso às notificações, botão de acesso rápido, ajuda e menu do usuário.
  • Área principal: Exibe o conteúdo da seção selecionada.

2.3. Tutorial Interativo

O CRM PEEEM possui um tutorial interativo que apresenta as principais funcionalidades do sistema. Para iniciá-lo:

  1. Clique no botão de ajuda (ícone de interrogação) na barra superior
  2. Selecione "Iniciar Tutorial" no menu de ajuda
  3. Alternativamente, clique no botão "Tutorial" na barra superior

O tutorial guiará você pelas principais seções do sistema, explicando suas funcionalidades.

Nota: Se o tutorial não iniciar automaticamente, verifique se seu navegador permite a execução de JavaScript e se cookies estão habilitados.

3. Painel de Controle

3.1. Visão Geral

A aba "Visão Geral" do painel de controle apresenta as principais informações do seu negócio:

  • Estatísticas: Visualize indicadores como faturamento diário, mensal, número de agendamentos e novos clientes.
  • Gráfico de receitas: Acompanhe a evolução da receita ao longo do tempo.
  • Serviços populares: Veja quais são os serviços mais procurados pelos clientes.
  • Despesas recentes: Visualize as últimas despesas registradas.
  • Aniversariantes do mês: Lista de clientes que fazem aniversário no mês atual.

3.2. Análises

A aba "Análises" apresenta informações mais detalhadas sobre o desempenho do seu negócio:

  • Taxa de retorno: Percentual de clientes que voltam a utilizar seus serviços.
  • Serviços por cliente: Média de serviços realizados por cliente.
  • Ticket médio: Valor médio gasto pelos clientes em cada visita.

3.3. Acesso Rápido

A aba "Acesso Rápido" permite acessar rapidamente as funções mais utilizadas do sistema:

  • Agendamentos: Criar novo agendamento ou consultar agendamentos existentes.
  • Financeiro: Registrar vendas, despesas ou visualizar relatórios.
  • Clientes: Cadastrar novo cliente ou gerenciar clientes existentes.
  • Serviços: Adicionar, listar ou editar serviços.

Também é possível acessar o menu de acesso rápido clicando no botão "Acesso Rápido" na barra superior.

4. Agendamentos

4.1. Criar Novo Agendamento

Para criar um novo agendamento:

  1. Acesse a seção "Agendamentos" no menu lateral
  2. Clique no botão "Novo Agendamento"
  3. Selecione o cliente (ou cadastre um novo)
  4. Escolha o serviço a ser realizado
  5. Selecione a data e horário
  6. Escolha o profissional responsável
  7. Adicione observações se necessário
  8. Clique em "Salvar"
Dica: Para agilizar o processo, você também pode criar um agendamento diretamente do painel de controle, na aba "Acesso Rápido".

4.2. Gerenciar Agendamentos

Para gerenciar os agendamentos existentes:

  1. Acesse a seção "Agendamentos" no menu lateral
  2. Visualize a lista de agendamentos
  3. Utilize os filtros para encontrar agendamentos específicos
  4. Clique em um agendamento para ver seus detalhes
  5. Use os botões de ação para:
    • Editar o agendamento
    • Cancelar o agendamento
    • Marcar como concluído
    • Reagendar

4.3. Visualizar Calendário

O calendário permite visualizar todos os agendamentos de forma organizada:

  1. Acesse a seção "Agendamentos" no menu lateral
  2. Clique na aba "Calendário"
  3. Escolha a visualização desejada:
    • Diária: visualize os agendamentos de um dia específico
    • Semanal: visualize os agendamentos da semana
    • Mensal: visualize os agendamentos do mês
  4. Clique em um horário vazio para criar um novo agendamento
  5. Clique em um agendamento existente para ver seus detalhes ou editá-lo
Dica: Use os filtros para visualizar apenas os agendamentos de um profissional específico ou de um tipo de serviço.

5. Financeiro

5.1. Registrar Venda

Para registrar uma nova venda:

  1. Acesse a seção "Financeiro" no menu lateral
  2. Clique no botão "Nova Venda"
  3. Selecione o cliente
  4. Adicione os produtos e/ou serviços vendidos
  5. Aplique descontos se necessário
  6. Selecione a forma de pagamento
  7. Clique em "Finalizar Venda"
Dica: Você também pode registrar uma venda a partir de um agendamento concluído, clicando no botão "Registrar Pagamento".

5.2. Registrar Despesa

Para registrar uma nova despesa:

  1. Acesse a seção "Financeiro" no menu lateral
  2. Clique no botão "Nova Despesa"
  3. Selecione a categoria da despesa
  4. Preencha o valor
  5. Adicione a descrição
  6. Selecione a data
  7. Anexe o comprovante (opcional)
  8. Clique em "Salvar"

5.3. Fechamento de Caixa

O fechamento de caixa deve ser realizado diariamente:

  1. Clique no botão "Fechamento de Caixa" na barra superior
  2. Confira o resumo das vendas e despesas do dia
  3. Verifique o valor em caixa
  4. Ajuste valores se necessário
  5. Adicione observações
  6. Clique em "Confirmar Fechamento"
Importante: O fechamento de caixa é essencial para manter a precisão dos relatórios financeiros. Recomendamos realizá-lo ao final de cada dia de trabalho.

5.4. Relatórios Financeiros

O sistema oferece diversos relatórios financeiros:

  • Relatório de Vendas: Visualize todas as vendas em um período específico.
  • Relatório de Despesas: Analise as despesas por categoria e período.
  • Fluxo de Caixa: Acompanhe as entradas e saídas do caixa.
  • Lucros e Perdas: Visualize o resultado financeiro do negócio.
  • Faturamento por Serviço: Identifique os serviços mais lucrativos.
  • Faturamento por Profissional: Analise o desempenho de cada profissional.

Para acessar os relatórios:

  1. Acesse a seção "Financeiro" no menu lateral
  2. Clique na aba "Relatórios"
  3. Selecione o tipo de relatório desejado
  4. Defina o período de análise
  5. Clique em "Gerar Relatório"
Dica: Você pode exportar os relatórios em formato PDF ou Excel para análise posterior ou para compartilhar com seu contador.
